torch_swiss.model¶

class torch_swiss.model.ModelHolder(model, should_clear_up_tmp=True)¶

ModelHolder is a model manager.

You can manual save model by using self.save_checkpoint,

and ModelHolder also auto save model while the end of ModelHolder.

All save model will store under ./.model_holder.

Usage Example:

` with ModelHolder(model) as (model_holder,model): `
